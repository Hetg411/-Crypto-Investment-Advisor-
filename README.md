# Crypto Investment Advisor

A web application that displays live cryptocurrency prices and helps users plan investments based on their salary, risk tolerance, and investment duration, using an ML model to predict future coin values.

## Features
- Live cryptocurrency price display
- Personalized investment planning based on salary, risk tolerance, and duration
- LSTM-based machine learning model to predict future cryptocurrency values and estimate investment returns
- Intuitive React interface with investment suggestions and insights into expected profits and coin allocation

## Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Backend/ML:** Python (LSTM model)

## Getting Started

### Prerequisites
- Node.js and npm installed
- Python 3.x with pip
- Required Python packages (e.g., `numpy`, `pandas`, `tensorflow`/`keras`) — see `requirements.txt`

### Installation

**Frontend:**
```bash
git clone https://github.com/Hetg411/-Crypto-Investment-Advisor-.git
cd -Crypto-Investment-Advisor-
npm install
```

**Backend (Python/ML):**
```bash
cd backend
pip install -r requirements.txt
```

### Run the App

**Backend:**
```bash
python app.py
```

**Frontend:**
```bash
npm start
```
Visit `http://localhost:3000` in your browser.

## Usage
1. Enter your salary, risk tolerance, and desired investment duration.
2. View live cryptocurrency prices.
3. Get ML-driven predictions on future coin values and expected returns.
4. Review personalized investment suggestions and coin allocation insights.
